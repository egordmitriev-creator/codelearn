# CodeLearn — Платформа обучения программированию

Веб-платформа для организации учебного процесса по программированию с автоматической проверкой кода.

---

## Быстрый старт (macOS / M1)

### 1. Требования (установить заранее)

| Что | Где скачать | Проверка |
|-----|-------------|----------|
| Python 3.9+ | https://python.org/downloads | `python3 --version` |
| Node.js 18+ | https://nodejs.org | `node --version` |

### 2. Запуск одной командой

```bash
cd codelearn
chmod +x start.sh
./start.sh
```

Скрипт сам:
- Установит все Python и Node зависимости
- Запустит бэкенд (порт 5000) и фронтенд (порт 3000)
- Откроет браузер на http://localhost:3000

### 3. Демо-аккаунты

| Роль | Логин | Пароль |
|------|-------|--------|
| 👨‍🏫 Преподаватель | `teacher` | `teacher123` |
| 🎓 Студент | `student` | `student123` |

---

## Возможности платформы

### Для преподавателя
- Создавать курсы
- Добавлять задания трёх типов: **код**, **тест** (один ответ), **вопрос**
- Для кодовых заданий — задавать открытые и **скрытые тест-кейсы** (ввод → ожидаемый вывод)
- Просматривать список студентов и их успеваемость
- Читать код, отправленный студентами

### Для студента
- Записываться на курсы
- Писать код прямо в браузере и **проверять до сдачи**
- Получать мгновенную обратную связь по каждому тесту
- Сдавать ответы после успешной проверки
- Отвечать на тестовые вопросы

## Структура проекта

```
codelearn/
├── backend/
│   ├── app.py          # Flask API + логика проверки кода
│   ├── requirements.txt
│   └── codelearn.db    # SQLite база (создаётся автоматически)
├── frontend/
│   ├── src/
│   │   ├── pages/      # Страницы приложения
│   │   ├── components/ # Переиспользуемые компоненты
│   │   └── context/    # Auth context
│   └── package.json
└── start.sh            # Скрипт запуска
```

## Технологии

- **Бэкенд**: Python + Flask + SQLite + JWT
- **Фронтенд**: React + Vite + React Router
- **Проверка кода**: subprocess (Python sandbox)
- **Аутентификация**: JWT токены

## Остановка

Нажмите `Ctrl+C` в терминале — скрипт остановит оба сервера.
